# Quiz Game

A simple and interactive web-based quiz game built with HTML, CSS, and JavaScript.

## Description

This project is a fun quiz application that allows users to test their knowledge on various topics. It features multiple-choice questions, score tracking, and a clean, responsive interface.

## How to Play

1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. Start the quiz by clicking the start button.
4. Answer each question by selecting an option.
5. View your final score at the end.

## Features

- Multiple-choice questions
- Score calculation
- Timer (if implemented in script.js)
- Responsive design for different screen sizes

## Technologies Used

- **HTML**: For the structure of the quiz interface
- **CSS**: For styling and layout
- **JavaScript**: For quiz logic, question handling, and interactivity

## Project Structure

- `index.html`: The main HTML file
- `style.css`: Stylesheet for the game
- `script.js`: JavaScript file containing the quiz logic
